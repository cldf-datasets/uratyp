**Can a 3Sg possessive suffix be used to mark definiteness?** 

Possessive suffixes may also occur in non-possessive functions, such as marking definiteness (see also UT104). This question asks whether 3Sg possessive suffixes can be used to convey definiteness. For example, in Nganasan, the 3Sg possessive suffix is used to indicate definiteness of celestial bodies and natural phenomena when the entity is a celestial body or when it is topicalized (Wagner-Nagy 2018:174). 

(1) Nganasan<br/>
>***kouðu** ŋǝðutɨ, ŋǝtumhi͡atɨ*<br/>
>kou-ðu ŋǝðu-tɨ, ŋǝtu-m-hi͡atɨ<br/>
>sun-3SGPOSS be.visible-AOR.3SG be.visible-TRL-INFER.3SG<br/>
>‘The sun is visible, it becomes visible’<br/>
(Wagner-Nagy 2019:174)

**Coding.** The value is '1' if a 3Sg possessive suffix can be used to mark definiteness. The answer is '0' if no such function can be attested or if possessive suffixes are not present in a language.

**References**<br/>
